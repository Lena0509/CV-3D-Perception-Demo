# CV-3D-Perception-Demo

This project demonstrates basic computer vision and 3D perception techniques, including:

## 🔹 Feature Matching
- ORB feature detection and matching
- Lowe's ratio test for filtering matches

## 🔹 Point Cloud Processing
- Point cloud visualization (Open3D)
- Downsampling and denoising

## 🔹 ICP Registration
- Point cloud alignment using ICP algorithm

## 🛠️ Tools
- Python
- OpenCV
- Open3D
- NumPy

## 📌 Usage
Run each module independently:

```bash
python feature_matching/test_cv1.py
python feature_matching/test_cv2.py
python point_cloud/3dcloud.py
python point_cloud/3dcloud1.py
python icp/ICP1.py
python icp/ICP1.py
